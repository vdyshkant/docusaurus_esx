## How to use 'dpkg' properly:

#### installation:
``` sudo dpkg -i ~/Downloads/stacer_*_amd64.deb; sudo apt -f install ```
or
``` sudo dpkg -i atom*.deb && sudo apt-get -f install ```

#### removing:
```
sudo apt-get remove --autoremove stacer
```
